Checkmarks - On & Off labels, Accept & Cross icons
    ✓ "On" labels
    Selecting an option in "On" status (accept icon) disables it. An "Off" status label and a cross icon appear.

    ✓ "Off" labels
    Selecting an option in "Off" status (cross icon) enables it. An "On" status label and an accept icon appear.
 
 
Settings button (bottom left)
    ✓ Settings guide

    ✓ Import settings from a text file

    ✓ Export settings to a text file

    ✓ Restore Noia 4 default skin and settings.
    All settings will be reset to default. In some cases a browser restart is needed and a confirmation window will appear after theme manager gets closed.
 
 
Customizeable tabs
 
If a tab skin label contains a "[customizable]" string, tabs can be styled by Noia 4s tab color settings or by extensions like TabMixPlus, TabUtilities, ColorfulTab or Favtabs.
