    Extra settings skin settings. Some settings are only visible, when the corresponding skin is selected.
     
    ✓ Rounded dropdown menus
    Popup and dropdown menus will have rounded corners.

    ✓ Remove blue color from hovered and active tabs
    Tabs will stay same color like the other tabs, when they are hovered or active (not compatible with settings and extensions, that changes tab colors).

    ✓ Remove tab toolbars background image
    Hides background image of the tabs toolbar. This is recommended for non-AeroGlass users, that are using tabs on top.

    ✓ Hide lines between toolbars
    Lines between toolabars disappear and give Firefox a cleaner look.

    ✓ Semi-transparent dropdown menus
    Popup and dropdown menus will have a semi transparent gray backgroung color.

    ✓ Black dropdown menus
    Popup and dropdown menus will have a black backgroung color.

    ✓ Glass dropdown menu
    Popup and dropdown menus will have AeroGlass effect.

    ✓ Less transparency on prefwindows & dialogs
    Prefwindows & dialogs will have less AeroGlass transparency.

Personas
 
Install and select a Persona → restart Firefox → enable Noia 4 theme inside Firefox Add-ons manager → restart Firefox → run Noia 4 theme manager → Skins tab → Personas (sub)tab and enable Personas
 
All steps have to be repeated, when another Persona is going to be used.
 
After closing Noia 4s theme manager a window asking for a browser restart will appear.
 
 
 
* 'Non-AeroGlass' users won't see this settings, because the default skin setting is equal to this one on 'Non-AeroGlass' operation systems.
