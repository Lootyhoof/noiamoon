FONTS TAB
 
Customize Noia 4s font colors and shadows.
 
Colors
    ✓ Tabs (normal)
    Text color on non-active and non-hovered tabs.

    ✓ Tabs (active/hove)
    Text color on active and hovered tabs.

    ✓ Toolbar
    Text color on almost all top toolbars.

    ✓ Status bar
    Text color on almost all bottom toolbars.

    ✓ Dropdown menu
    Text color on dropdown and popup menus.
 
 
Shadows
    ✓ Tabs (normal)
    Text shadow color on non-active and non-hovered tabs.

    ✓ Tabs (active/hove)
    Text shadow color on active and hovered tabs.

    ✓ Toolbar
    Text shadow color on almost all top toolbars.

    ✓ Status bar
    Text shadow color on almost all bottom toolbars.

    ✓ Dropdown menu
    Text shadow color on dropdown and popup menus.
 
 
Font settings will be disabled on skin change. They will be enabled and set to white automatically, when a black skin is activated. It is possible to change colors afterwards.
 
Tab color settings are not compatible with 'ColorfulTabs extension' and other extensions, that change tab font colors.
