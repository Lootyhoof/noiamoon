TWEAKS TAB - page 1
 
Firefox appbutton
 
This settings are only for Windows and Linux systems!
    ✓ Default (Noia 4)
    Appbutton is transparent.

    ✓ Orange
    Appbutton has finals/betas orange color.

    ✓ Aurora
    Appbutton has Auroras dark blue color.

    ✓ Nightly
    Appbutton has Nightlys black color.

    ✓ Always orange
    Appbutton has always finals/betas orange color in normal and private browsing mode.

    ✓ Always transparent
    Appbutton has always transparent background in normal and private browsing mode.

    ✓ Display icon only
    Show a white Firefox icon instead of the 'Firefox' text string.

    ✓ Display icon and text
    Show a white Firefox icon and the 'Firefox' text string.

    ✓ Remove dropmarker
    Remove the white dropmarker arrow from appbutton.
 
Buttons (on preferences/dialogs)
    ✓ Dark buttons
    Dark buttons (mainly) for black skins.

    ✓ Transparent buttons
    Transparent buttons (mainly) for AeroGlass skins.

    ✓ Partly rounded buttons
    Buttons with partly rounded corners like on Firefox default theme.
 
Scrollbars
    ✓ Default (Noia 4)
    Themes default scrollbar appearance.

    ✓ Dark (Noia 4)
    Themes default scrollbar appearance (mainly) for black skins.

    ✓ Firefox default or from "NewScrollbars" extension
    Use Firefox default gray scrollbars to prevent bugs on some systems or style scrollbars with "NoiaScrollbars" extension.
Every website has to be reloaded after changing scrollbar settings.
 
TWEAKS TAB - Tabs
 
Tab options
Some settings may be incompatible with other tab extensions.
    ✓ Bigger tab height (restart Firefox recommended)
    Tabs toolbar gets a greater height. Restart Firefox is recommended, if multi-row tabs are used.

    ✓ Smaller tab height (restart Firefox recommended)
    Tabs toolbar gets a smaller height. Restart Firefox is recommended, if multi-row tabs are used.

    ✓ Alternative bigger tab close-button
    Replaces default tab close button with a bigger one.

    ✓ Hide tab close-button on non-focussed tabs
    Tab close-button is only visible on selected and hovered tabs. This setting is not compatible with TabMixPlus setting "Tab width fits to tab title".

    ✓ Alternative close-button
    Replaces default tab close button.

    ✓ Always show close-button (on tabs 2-7)
    Shows always close-tab button on tabs 2-7. Because of Firefox restritions it is not possible to show it on more than seven tabs.

    ✓ Combine favicon and close tab icon
    Close-tab icon and favicon now share same place.

    ✓ Rounded tabs (not for Firefox 4+ skins)
    All tab cornes get rounded. Since tab corners on "Firefox W7" skins already have a different roundness, it is not recommended to use this setting along with them.

    ✓ Halfrounded upper tab corners
    Upper tab cornes get rounded like on "Firefox W7" skins.

    ✓ Remove bold font from hovered tab
    Normal font weight on hovered tabs.

    ✓ Remove bold font from active tab
    Normal font weight on active tabs.

    ✓ More space between icons and tab text
    Adds more space between tab image and tab text and between tab text and close image.

    ✓ Blinking active tab [v1]
    Slow blinking active tab (red-green).

    ✓ Blinking active tab [v2]
    Fast blinking active tab (many colors).

    ✓ Blinking active tab [v3]
    Slow blinking active tab (many colors).

    ✓ Alternative tab throbber (loading animation) [v1]
    Tab loading animation like on default Firefox theme.

    ✓ Alternative tab throbber (loading animation) [v2]
    Colorful orb as tab loading animation.

    ✓ Alternative tab progress indicator for TabMixPlus [v1]

    ✓ Alternative tab progress indicator for TabMixPlus [v2]

    ✓ Alternative tab progress indicator for TabMixPlus [v3]

TWEAKS TAB - Toolbar
 
Navigation toolbar settings
    ✓ Firefox 4-12 buttons [v1]
    Default Firefox 4-12 toolbar buttons with dark toolbar button icons.

    ✓ Firefox 4-12 buttons [v2]
    Default Firefox 4-12 toolbar buttons with white toolbar button icons.

    ✓ Smaller buttons (Noia)
    Toolbar button icon sizes get inverted (normal ↔ hover) and look smaller.

    ✓ Gray buttons
    All toolbar button icons are stripped from color and displayed in levels of gray.

    ✓ Smaller forward button [v1]
    Forward button size is reduced and space between backward button and forward button is reduced like on default Firefox theme.

    ✓ Smaller forward button [v2] (no overlapping)
    Forward button size is reduced and the space between backward button and forward button is untouched.

    ✓ Smaller forward button [v3]
    Forward button size is reduced even more and the space between backward buttonand forward button is reduced like on default Firefox theme.

    ✓ Lightish color behind hovered buttons
    Hovered toolbar buttons gain a silver-gray gradient background.

    ✓ Blue color behind hovered buttons
    Hovered toolbar buttons gain a blue glowing background.

    ✓ No background on "checked/active" nav-bar buttons
    This setting removes the 'pressed' background color effect from toolbar buttons like 'history button' or 'bookmarks button'.
 
Recent Pages Dropmaker (aka back-forward-dropmarker)
    ✓ Recent Pages Dropmaker [v1]
    Dropmarker appears right next to the forward button like it was on Firefox 3.

    ✓ Recent Pages Dropmaker [v2]
    Dropmarker appears between forward and backward button. This setting is not compatible with Firefox 4-12 toolbar button settings.
 
 
TWEAKS TAB - Misc
 
Options (miscellaneous)
    ✓ Bigger font on menubar
    Larger and bolder font size for menubar text (File, Edit, View...).

    ✓ Higher bookmarks toolbar
    Larger height for bookmarks toolbar.

    ✓ Bigger font on urlbar and searchbar
    Larger font size for text on adress-/urlbar and searchbar.

    ✓ Bold font for popup menu text labels
    Bold font for popup menu items.

    ✓ Compact popup menus
    Less space between entries on popup menus.

    ✓ Remove close-button from add-on bar
    Hide add-on toolbars close button.

    ✓ Remove close-button from findbar
    Hide findbars close button.

    ✓ Purple bookmarks folder
    Replace blue bookmarks folders with purple bookmarks folders.

    ✓ Rotate icons 15deg on hover
    Almost all icons will rotate 15deg on mouse over.

    ✓ Rotate icons 360deg on hover
    Almost all icons will rotate 360deg on mouse over.

    ✓ Tooltip appearance like on "Windows XP"
    The default yellow tooltip is shown when items are hovered (like on Windows XP).

    ✓ MacOSX style window control icons
    MacOS-style for minimize, maximize, restore and close buttons on Firefox main window. The effect of this setting is only visible on Windwos XP and Windows 7/Vista with AeroBasic GUI and not visible on Windows 7/Vista with AeroGlass GUI or other operation systems..

    ✓ Noia4 style window control icons
    Noia4-style for minimize, maximize, restore and close buttons on Firefox main window. The effect of this setting is only visible on Windwos XP and Windows 7/Vista with AeroBasic GUI and not visible on Windows 7/Vista with AeroGlass GUI or other operation systems.

    ✓ Blue background color (1)
    Default blue background color. Differs between dark and light skins.

    ✓ Blue background color (2)
    Stronger blue background color and white font for hovered and active menu items.

    ✓ Purple background color
    Purple background color and white font for hovered and active menu items.

    ✓ Semitransparent background
    Semitransparent white background color for hovered and active menu items.

    ✓ Transparent background
    No background color for hovered and active menu items.
