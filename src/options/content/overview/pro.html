'Pro' TAB
 
Favicon
 
    ✓ Agreement
    It is important the user has read, understood and accepted the terms of use before the favicon option can be used.

    ✓ Restore old 'Favicon in urlbar'
    This will place the tab image (favicon) back into the urlbar like it was on older Firefox versions. After changing this option Firefox needs to be restated.

    ✓ Hide globe icon for 'Favicon in urlbar'
    Hides the globe icon, that is actally not needed anymore after enabling the favicon option. After changing this option Firefox needs to be restated.
 
 
Firefox internal settings (about:config)
 
    ✓ Tabs on Top
    The known Firefox option disappears from Firefox GUI once Firefox is restarted with tabs "on top". This setting switched between "on top" and "old default" and be changed on the Pro-tab without using the about:config area.

    ✓ Classic downloads-button appearance
    Restore the old downloads-button without progress indicator. Download window opens once a download is started (like on older Firefox versions). This setting can be changed on the Pro-tab without using the about:config area.

    ✓ Preferences in Content
    This new feature may be introduced in future Firefox versions. Users, that prefer the preferences window can enable/disable it on "Pro"-tab without using the about:config area.
This settings don't get changed by theme managers import, export and restore features.
