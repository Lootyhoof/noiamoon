Help (English) / Aide (français)

Noia Moon is a package of:

    4 themes for Pale Moon: grey, light grey, blue and black.

    Noia Moon options: An add-on with multi-options. Customize Noia Moon as you want.

 

Noia Moon Options

 

Warning

    When you select a new option, sometimes you can see graphic bugs.

    It is the reason why you must restart Pale Moon after selecting a new option.

    You can also restart Pale Moon when you maximize or normalize the window if you see graphic bugs.

    Another option is to reset all Noia Moon settings: go to 'display' tab - 'Advanced tab'

    and push the 'Restore Noia Moon default settings' button.

 

Grey theme

    All theme skinned.

 

Light grey theme

    Some elements are skinned like the first grey theme (options, add-ons,....).

 

Blue theme

    Some elements are skinned like the first grey theme (options, add-ons,....).

 

Black theme

    Full dark theme.

    Personas don't work with this theme.

 

Square tabs, Semi rounded tabs and Rounded tabs

    Tabs display. Original display is square tabs.

 

Tabs with no shadow

    If enabled, tabs will be displayed with no shadow (inside and outside the tab).

    You can use it with add-ons that don't support tabs with shadows.

    If enabled with 'Transparent tab', tab will show only a border.

 

Solid tabs / Transparent tabs

    For Grey theme and black theme.

    Pale Moon button is also modified (grey theme).

    Checked: Solid tabs / Unchecked : Transparent tabs

    If 'Solid tabs' is enabled, customised tabs cannot be used with Tab Mix Plus.

 

Tabs at the bottom of the window

    Tab groups button, List all tabs button and Tab Mix Plus close tab button are hidden.

    If you want these buttons, it's better to use the TabMixPlus add-on.

    Download statusbar add-on : switch to Mini Mode before using this option.

    This option is deactivated when you activate 'Auto hide toolbars' option (and vice versa).

 

Tabbar background transparent

    Useful with Personas. If selected, all is skinned with the Persona.

    Not available with 'Tabs at the bottom of the window' option.

    (Grey theme only).

 

Tabbar Bottom Strip

    Add a blue tabbar bottom strip.

    (Grey theme and blue theme only).

    You must disable this option if you want all features of the ColorfulTabs add-on.

 

Alternative blue close button

    All close buttons (red) are replaced with a blue Mac button.

 

No border for Toolbars

    Borders/no borders tweak for the toolbars. Old Noia 2 extreme option.

    Borders/no borders for tabs toolbar (except with tabs under the navigation toolbar with original tabs toolbar background).

    If you don't want borders with the black theme, first choose black theme, then choose "No border".

    Restart Pale Moon (Borders are included with the black theme)

    Exception: Yahoo toolbar will always come with borders.

 

Alternative grey theme titlebar

    Titlebar display like versions before Noia Fox 1.4.4.

    With tabs on top and tabs under the navigation toolbar.

 

Original Windows titlebar (Windows OS only)

    Checked - Original Windows titlebar (except : tabs in titlebar -> titlebar is still skinned).

    Unchecked - Titlebar is skinned, with Mac caption buttons.

    Mac caption buttons are always displayed with Personas.

 

Alternative grey theme background

    Tabs in titlebar : lighter background.

    Tabs not in titlebar : darker background.

 

Auto hide toolbars

    When Auto hide option is active, all toolbars are hidden. You can only see the tabs and the menubar (if not hidden with Pale Moon option).

    Hovering over tabs will reveal the bars.

 

Square/Rounded buttons

    Grey theme only.

 

Dialog boxes: Square/Rounded tabs

    Grey theme only.

 

Noia warm icons (except: folders and bookmarks)

    Second icon set created by Carlitus.

 

Noia warm icons for folders and bookmarks

    Unchecked: original blue folders and bookmarks icons.

    Checked: Red folders and bookmarks icons (warm icons).

 

Blue/warm mixed folders and bookmarks

    Unchecked: original blue folders and bookmarks icons.

    Checked: Red folders and bookmarks icons (warm icons) when hovering the icons.

 

Bookmarks toolbar

    Two options:

    - Icons only.

    - Icons and text.

    You can select these options when Pale Moon toolbar option is 'Icons' or 'Icons and text'.

 

Add-ons display in the navbar

    Two options:

    - Icons only.

    - Icons and text.

    You can select these options only when Pale Moon toolbar option is 'Icons and text'.

 

Bookmarks toolbar buttons display when hovering

    Two options:

    - No background.

    - With background.

 

Addons: Toolbar buttons display when hovering

    Two options:

    - No background.

    - With background.

    Supported toolbars: AddThis, Amazon, FoxLingo, Googlebar lite, Torrent Finder, Webdeveloper, Webmail Notifier.

 

Navigation toolbar buttons display when hovering

    Two options:

    - No background.

    - With background.

 

Persona enable

    See the Personas help file.

 

Add-ons Limitations

'Colorful tabs' add-on

    You must disable 'Tabbar Bottom Strip' option if you want all features of the ColorfulTabs add-on.

 

Pale Moon Status Bar

    If you have a progress bar bug in the location bar (Pale Moon Status Bar + Locationbar 2 add-on) :

    Enable Status Bar Options -> Advanced -> Attempt to force the Location Bar XBL binding

 

Noia Moon improvements

    If you want Tabs in titlebar with normal window, you can install the 'Tabs always in titlebar' add-on:

    https://addons.mozilla.org/en-US/firefox/addon/tabs-always-in-titlebar/

    You must restart Pale Moon when you enable 'Tabs always in titlebar' add-on for best graphic display with Noia Moon.

 

    If you want to move Status Bar Icons, Pale Moon Menu Button and Menu Bar to any toolbar + Page Title in Titlebar display,

    you can install the 'UI Fixer' add-on: https://addons.mozilla.org/en-US/firefox/addon/firefox-4-ui-fixer/

 

    If you want to add a lot of customizable visual styles to Personas themes, you can install the 'Personas Expression' add-on:

    https://addons.mozilla.org/en-US/firefox/addon/personas-expression/

    - Modify color, shadow and font width of texts.

    - Modify color of tabs.

    - Modify transparency of objects.

    - Draw selected color and texture on background of objects.

    - Draw shadows on bottom of title-bar. (In Mac, title-bar color will be modified.)

    - Display Personas theme on some pages. Empty page, image page and more.

    - Modify width of tabs.

    - Modify height of side-bar title.

    - Transparent status-bar to improve visibility of Personas footer. (Windows Classic theme)

    - Easy access to Personas Gallery.

Noia Moon est un package constitué par :

    4 thèmes pour Pale Moon : gris, gris clair, bleu et noir.

    Noia Moon options : Un module complémentaire multi-options permettant de personnaliser Noia Moon.

 

Noia Moon Options

 

Attention

    Quand vous selectionnez une nouvelle option, vous pouvez voir de temps en temps des bugs graphiques.

    (essentiellement lors du passage du thème gris au thème noir).

    C'est la raison pour laquelle vous devez redémarrer Pale Moon après avoir choisi une nouvelle option.

    Vous pouvez également redémarrer Pale Moon quand vous maximisez / normalisez la fenêtre si vous voyez des bugs graphiques.

    Une autre option est de réinitialiser tous les réglages de Noia Moon : allez dans l'onglet 'Apparence' - onglet 'Avancé'

    et appuyez sur le bouton 'Restaurer les réglage par défaut de Noia Moon'.

 

Thème gris

    Theme complet.

 

Thème gris clair.

    Certains éléments sont habillés comme le premier thème gris (options, modules complémenataires,....).

 

Thème bleu

    Certains éléments sont habillés comme le premier thème gris (options, modules complémenataires,....).

 

Thème noir : second thème.

    Tout le thème est relooké en noir.

    Les Personas ne fonctionnent pas si cette option est sur Oui.

 

Onglets carrés, onglets demi-ronds, onglets ronds

    Aspect des onglets. Par défaut les onglets sont carrés.

 

Onglets sans ombre

    Si cette option est selectionnée, les onglets apparaissent sans ombre (à l'interieur et à l'exterieur de l'onglet).

    Vous pouvez l'utiliser avec les modules ne supportant pas les ombres pour les onglets.

    Si cette option est couplée à l'option 'Onglets transparents', les onglets n'apparaitront qu'avec une bordure.

 

Onglets pleins / Onglets transparents

    Pour le thème gris et le thème noir.

    Le bouton Pale Moon est également modifié (thème gris).

    Selectionné : Onglets pleins / Deselectionné : Onglets transparents

    Si 'Onglets pleins' est selectionné, les onglets customisés ne peuvent pas être utilisés avec Tab Mix Plus.

 

Onglets en bas de la fenêtre

    Les boutons 'Lister tous les onglets', 'Organiser les onglets par groupe' et le bouton 'Fermer l'onglet' de Tab Mix Plus sont desactivés.

    Il est préférable d'utiliser le module Tab Mix Plus si vous voulez tous ces boutons.

    Module complémentaire Download Statusbar : passez en Mini Mode avant d'activer cette option.

    Cette option est inactivée si l'option 'Masquer automatiquement les barres d'outils' est activée et inversement.

 

Fond de barre d'onglet transparent

    Utile lorsque vous installez un Personna. Si l'option est mise sur oui tout est habillé par le Personna.

    Non disponible avec l'option 'Onglets en bas de la fenêtre'.

    (Thème gris uniquement).

 

Bande en bas le la barre d'onglets

    Ajoute bande bleue en bas de la barre d'onglets.

    (Thèmes gris et bleu uniquement).

    Vous devez désactiver cette option si vous voulez utiliser toutes les possibilités du module ColorfulTabs.

 

Bouton de fermeture bleu

    Tous les boutons de fermeture (croix rouge) sont remplacés par un bouton bleu style Mac.

 

Pas de bord pour les barres

    Tweak pour les barres d'outils. C'est une ancienne option de Noia 2 extreme.

    Valable pour la barre d'onglets (excepté lorsque les onglets sont sous la barre de navigation avec le fond original de la barre d'onglets).

    Si vous ne voulez pas de bord pour le thème noir, chosissez d'abord le thème noir, puis choisissez 'Pas de bord'.

    Redémarrez Pale Moon (Le thème noir est d'origine avec des bords).

    Exception : La barre Yahoo aura toujours des bords.

 

Barre de titre alternative pour le thème gris

    La barre de titre est comme celle des versions avant Noia Fox 1.4.4.

    Avec la barre des onglets au dessus et en dessous de la barre de navigation.

 

Barre de titre originale de Windows (Système d'exploitation Windows seulement).

    Selectionné : Barre de titre originale de Windows (sauf avec les onglets dans la barre de titre --> la barre de titre reste habillée).

    Deselectionné : La barre de titre est habillée, avec les boutons style Mac (boutons minimiser, agrandir et fermer Pale Moon).

    Les boutons style Mac resteront affichés avec les Personas.

 

Fond alternatif pour le thème gris.

    Onglets dans la barre de titre : fond plus clair.

    Autres cas : fond plus sombre.

 

Masquer automatiquement les barres d'outils

    Quand l'option de masquage est activée, toutes les barres d'outils sont masquées. Seuls les onglets sont visibles

    et la barre de menu (si elle n'est pas masquée par l'option Pale Moon).

    Lorsque vous survolez avec la souris les onglets, les barres apparaissent.

 

Boutons carrés/arrondis

    Thème gris seulement.

 

Onglets des boites de dialogues carrés/arrondis

    Thème gris seulement.

 

Icônes Noia Warm (sauf dossiers et favoris)

    Deuxième set d'icônes réalisé par Carlitus. Le rendu est excellent avec le thème noir.

 

Dossiers et favoris : icônes Noia Warm

    Deselectionné : dossiers et favoris originaux (bleus).

    Selectionné : Dossiers et favoris rouges (icônes warm).

 

Couleur des dossiers et favoris mixée (bleu/warm)

    Deselectionné : dossiers et favoris originaux (bleus).

    Selectionné : Dossiers et favoris rouges (icônes warm) lorsque vous survolez les icônes.

 

Barre personnelle

    2 options :

    - Icône seule.

    - Texte et icône.

    Quelque soit l'option des barres d'outils de Pale Moon, vous pouvez

    choisir d'afficher 'uniquement les icônes' ou les 'texte et icônes' pour la barre personnelle.

 

Aspect des modules dans la barre de navigation

    2 options :

    - Icône seule.

    - Texte et icône.

    Uniquement lorsque l'option des barres d'outil de Pale Moon est sur 'Texte et icônes'.

 

Aspect des boutons de la barre personnelle lors du suvol de ceux-ci

    2 options :

    - Sans arrière-plan.

    - Avec arrière-plan.

 

Modules : Aspect des boutons des barres d'outils lors du suvol de ceux-ci

    2 options :

    - Sans arrière-plan.

    - Avec arrière-plan.

    Barres d'outils supportées : AddThis, Amazon, FoxLingo, Googlebar lite, Torrent Finder, Webdeveloper, Webmail Notifier.

 

Aspect des boutons de la barre de navigation lors du suvol de ceux-ci

    2 options :

    - Sans arrière-plan.

    - Avec arrière-plan.

 

Activer les Personas

    Consultez le fichier d'aide des Personas.

 

Limitations pour certains modules

Module 'Colorful tabs'

    Vous devez desactiver l'option 'Bande en bas le la barre d'onglets' si vous voulez utiliser toutes les possibilités du module ColorfulTabs.

 

Pale Moon barre d'état

    Si vous avez un bug de la barre de progression dans la barre de navigation (Pale Moon barre d'état + module Locationbar 2) :

    Options de la barre d'état -> Avancés -> Cochez 'Tentative de forcer le binding XBL de la barre de navigation'.

 

Améliorations de Noia Moon

    Si vous voulez les onglets dans la barre de titre avec fenêtre normale, installez le module 'Tabs always in titlebar' :

    https://addons.mozilla.org/en-US/firefox/addon/tabs-always-in-titlebar/

    Vous devez redémarrer Pale Moon après avoir activé le module 'Tabs always in titlebar' pour un meilleur rendu avec Noia Moon.

 

    Si vous voulez mettre les icônes de la Barre d'état, le bouton Pale Moon et la barre de Menu dans n'importe quelle barre d'outils

    + le Titre de la Page dans la barre de titre, installez le module 'UI Fixer' : https://addons.mozilla.org/en-US/firefox/addon/firefox-4-ui-fixer/

 

    Si vous voulez ajouter plein d'effets visuels aux Personas (couleurs des textes, des onglets, fond, transparence,...), installez le module

    'Personas Expression' :

    https://addons.mozilla.org/en-US/firefox/addon/personas-expression/
